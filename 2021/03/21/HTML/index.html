HTML:基本语法、常用标签等

HTML

0. Introduction

  • HTML: Hyper Text Markup Language 超文本标记语言

    HTML不是编程语言,是标记语言,由标签构成;使用标记标签来描述网页

  • 语法规范:

    • 标签分为开始标签，结束 便签和自闭合便签.如<html></html> <br />

    • 标签可以嵌套，但是需要注意嵌套的格式，不能非法嵌套;

      Error:<a><b></a></b>

    • HTML标签不区分大小写，建议使用小写。
    • 可以在开始标签中定义属性(键值对)，值需要用引号引起来。
    • 注释：<!-- 需要注释的内容 -->

  • 基本格式：

    <html>
    <head>
        <title>Your title</title>
    </head>
    
    <body>

    </body>
</html>

  • HTML常用特殊字符:

    • &nbsp:空格字符，若直接在HTML中输入空格，无论输入多少个空格也只能展示出一个空格。若想展示出多个空格必须使用&nbsp特殊字符来表示
    • &amp:表示字符&和
    • &lt:表示字符<小于号
    • &gt:表示字符>大于号

  • HTML<!DOCTYPE>声明:

    声明有助于浏览器中正确显示网页。网络上有很多不同的文件，如果能够正确声明HTML的版本，浏览器就能正确显示网页内容。

    在HTML 5中直接<!DOCTYPE html>即可。

1. HTML基本标签

1.1 标题

<h1> </h1>
<!-- html中标题从h1到h6字体大小逐级递减 -->
<!-- 浏览器会自动地在标题前后添加一个空行 -->

请勿为了加粗字体来设置标题,索引目录是依据标题来呈现文章结构.

1.2 段落

<p> </p>

1.3 链接

<a href = "url"></a>

a标签常用属性：

* href：指定访问的url
  • target：指定打开资源的方式
    • _blank:在空白页打开
    • _self:在当前页打开 默认值

1.4 换行

<br />
<!-- 换行是自结束标签 -->

1.5 水平线

<hr />
<!-- 水平线是自结束标签 -->
<hr color = "red" align = "center" />

常用属性:

* color:颜色
* width:宽度
* size:高度
* align:对齐方式("center" "right" "left" )

1.6 字体

<p>
    <b>这是加粗字体</b>
    <br />
    <i>这是斜体</i>
    <br />
    <font color = "red ">
        我变红了
        <center>我要居中显示</center>
    
    </font>
</p>

font标签常用属性

* color:颜色
* size:字体大小
* face:指定字体

color属性的值可设置为以下几种：

  • 颜色名:仅有16种颜色名被W3C的HTML 4.0标准所支持，它们是：aqua，black，blue，fuchsia，gray，green，lime，maroon，navy，olive，purple，red，silver，teal，white，yellow。

  • 颜色值:颜色由一个16进制符号来定义,这个符号由红色绿色蓝色的值组成(RGB);

    每种颜色的最小值为0(00),最大值为255(##);

    • rgb(value1,value2,value3)
    • rgb(#00FF00)

1.7 图像

<img src = "url" />

img标签常用属性:

* src: source 即资源路径
* alt: alter 用于定义一串预备的可替换的文本，当无法正常载入图像时则会显示alt的value
* width/height:宽度和高度 单位是像素px

1.8 列表

  • 列表分为有序列表和无序列表；

  • 有序列表(Ordered List)：使用数字进行标记

<ol>
    <li>第一</li>
    <li>第二</li>
    <li>第三</li>
</ol>
  • 无序列表(Unordered List)：使用小黑点进行标记
<ul>
    <li>第一</li>
    <li>第二</li>
</ul>

1.9 表格

  • 表格由<table>标签来定义，每个表格有若干行，每行以<tr>( Table Row)来定义。每行被分为若干单元格，单元格由<td>(Table Data)来定义。数据单元格可包含文本图片链接等多种内容。

  • 表格的表头单元格会使用<th>来定义，大多数浏览器会把表头显示为粗体居中的文本。

  • 表格的标题可以使用<caption>标签来定义,还可以使用<thead>``<tbody>``<tfoot>来定义表格的头部分,体部分和底部分，从而改变表格结构。

<table border = “1”>
    <th>name</th>
    <th>salary</th>
    <tr>
        <td>zhangsan</td>
        <td>100</td>
    </tr>
    <tr>
        <td>lisi</td>
        <td>200</td>
    </tr>
</table>

table标签常用属性

* border:边框属性 
* width:宽度
* bgcolor:背景颜色
* align:对齐方式

合并单元格:

​ 标签中有colspan(合并列)和rowspan(合并行)属性，value为需要合并的个数。合并按照从左到右，从上到下的顺序来。

1.10 块级元素与内联元素

  • 在HTML里，可以通过<div>或<span>标签将HTML元素组合起来。

  • 块级元素(Blocked Level Element)在浏览器显示时,通常会以新行来开始和结束。eg:<h1>,<p>,<table>

  • 内联元素(Inline Element)在显示时通常不会以新行为开始.。eg:<b>``<a>``<img>

  • <div>标签元素是块级元素,与CSS一起使用时<div>可用于对大的内容块设置样式属性.

  • <span>标签元素是内联元素,与CSS一起使用时<span>可用于对部分文本设置样式属性.

1.11 <iframe>框架标签

  • 使用<iframe>标签可以在html界面上打开一个内嵌小窗口.

  • <iframe>和 <a> 标签组合使用的步骤:

    ​ 1.在 <iframe> 标签中使用 name 属性定义一个名称

    ​ 2.在 <a>标签的 target 属性上设置 <iframe> 的 name 的属性值

2. HTML 表单标签

2.1 概述

  • 表单标签用于采集用户输入的数据,将其与服务器进行交互.
  • HTML中使用<form></form>来定义表单

2.2 <form>标签中的属性

  • action：指定提交数据的url

  • method：指定提交方式，主要使用的有以下两种

    • GET：默认提交方式

      • 请求参数会在地址栏中显示，封装在请求行中。
      • 请求参数的大小有限制。
      • 相对不安全。

    • POST：

      • 请求参数不会在地址栏中显示，封装在请求体中。
      • 从理论上来说，请求参数的大小无限制。
      • 相对安全。

2.3 <input>标签

  • 可以通过改变type属性的值来改变元素展示的样式.

  • type属性值:

    • text：文本输入框，Default。value设置默认值。

    • password：密码输入框，输入时已输入的字符用小黑点表示。value设置默认值。

    • radio：单选框。可以使用name属性对其进行分组，对一个分组内的实现单选的效果。若想设置默认勾选，可以给checked属性赋值checked从而表示默认选中。

    • checkbox：复选框。同上。

    • file：文件选择框。

    • hidden：隐藏域，用于提交一些信息。（当我们要发送某些信息，而这些信息，不需要用户参与，就可以使用隐藏域，提交的时候同时发送给服务器 ）

    • submit：提交按钮，可以用于提交表单。value属性表示按钮上的文本。

    • button：普通按钮。value属性表示按钮上的文本。

    • reset：重置按钮。value属性表示按钮上的文本。

    • image：图片提交按钮。src属性指定图片路径。

  • label属性:

    <label> 标签为 input 元素定义标注（标记）。

    label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上。

    <label> 标签的 for 属性应当与相关元素的 id 属性相同。

  • placeholder属性值：指定输入框的提示信息，当输入框的内容发生变化，会自动清空提示信息

2.4 其他常用标签

  • <select>标签:下拉列表框,与<option>标签配合使用.<option>标签是下拉列表中的选项.当selected属性为selected时则为默认选中
  • <textarea>标签:多行文本输入框.通过rows属性设置可以显示几行的高度,通过cols属性设置每行可以显示几个字符宽度.其中其开始标签和结束标签中间的内容为默认值.

2.5 案例

<!-- 需求:创建一个个人信息注册的表单界面。包含用户名，密码，确认密码。性别（单选），兴趣爱好（多选），国籍（下拉列表）。 隐藏域，自我评价（多行文本域）。重置，提交 -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <p>登录界面</p>
    <form>
        用户名:<input type="text" placeholder="请输入您的用户名" />
        <br/>
        密码:<input type="password" placeholder="请输入您的密码" />
        <br/>
        确认密码:<input type="password" placeholder="请再次确认您的密码" />
        <br/>
        性别:<input type="radio" name="sex">男 <input type="radio" name="sex">女
        <br/>
        兴趣爱好:<input type="checkbox" name="hobbies" checked="checked">编程 <input type="checkbox" name="hobbies"> 跑步 <input type="checkbox" name="hobbies">唱歌
        <br/>
        国籍:<select>
            <option>--请选择国籍--</option>
            <option>中国</option>
            <option>美国</option>
            <option>俄罗斯</option>
        </select>
        <br/>
        <input type="hidden" value="hidden---">
        自我评价:<textarea rows="10" cols="20">我的自我评价:</textarea>
        <br/>
        <input type="submit" value="提交">
        <input type="reset" value="重置">
    </form>
</body>
</html>
