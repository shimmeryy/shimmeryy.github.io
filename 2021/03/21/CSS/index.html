CSS基础部分：基本语法、与HTML结合方式、选择器、常用标签属性

CSS基础

0. Introduction

  • CSS： Cascading Style Sheets(层叠样式表/级联样式表) ，描述了网页的布局。

  • 基本语法：

    • 格式:

      选择器{
    key1:value1;
    key2:value2;
    ...
}

    • 注释:/* */

1. CSS与HTML的结合方式

1.1 内联样式

  • 在标签的style属性中设置”key:value”，从而改变标签的样式。

eg:

<p style="color:red;">test!!</p>
  • 缺点:代码重复度高,不适合用于实际开发环境中

1.2 内部样式

  • 在<head></head>标签中使用style标签来定义自己需要的CSS样式。

eg:

<style type="text/css">
    p{
        color:green;
    }
    </style>
  • 缺点:只能在同一页面复用代码，不能在多个页面中复用CSS代码，难以维护。

1.3 外部样式

  • 在<head></head>标签中通过<link>标签引入CSS样式文件。

  • <link>标签属性:

    • rel(relationship)：规定链接类型,指当前文档与被链接文档之间的关系。在与CSS结合时使用stylesheet这个属性值。
    • type:：指定所连接文档的MIME类型，css的MIME是type/css,一般使用type=”text/css”。
    • href：定义资源的链接 URL。内容为一个绝对URL链接或者是相对URL。

eg：

p{
    color:red;
}
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" type="text/css" href="./style.css"/>
</head>
<body>
    <p>test!!</p>
</body>
</html>

2. CSS选择器

2.1 id选择器

  • 选择一个具体的id属性值的元素
  • 语法:#id属性值{} (注意id属性值不能以数字开头,且建议在html里id值唯一)

eg:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #p02{
            color: red;
        }
    </style>
</head>
<body>
    <p id="p01">test01</p>
    <p id="p02">test02</p>
</body>
</html>

2.2 元素选择器

  • 选择具有相同标签名称的选择器

eg:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        p{
            color: red;
        }
    </style>
</head>
<body>
    <p id="p01">test01</p>
    <p id="p02">test02</p>
</body>
</html>

2.3 类选择器

  • 选择具有相同class属性值的元素.
  • 语法:.class属性值{}

eg:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .p2{
            color: red;
        }
    </style>
</head>
<body>
    <p class="p1">test01</p>
    <p class="p2">test02</p>
</body>
</html>

2.4 组合选择器

  • 格式:

    选择器1,选择器2,...,选择器n{
	key:value;
}

  • 使用组合选择器可以让多个选择器共同一段CSS样式代码。

eg:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .p1,#p2{
            color: red;
        }
    </style>
</head>
<body>
    <p class="p1">test01</p>
    <p id="p2">test02</p>
</body>
</html>

2.5 扩展选择器

  • 选择所有元素: *{} 使用*通配符

  • 子选择器:筛选选择器1元素下的选择器2中的元素

    语法:选择器1 选择器2 {}

  • 父选择器:筛选选择器2的父元素选择器1中的元素

    语法:选择器1 > 选择器2{}

  • 属性选择器:选择元素名称,key=value的元素

    语法:元素名称[属性名="属性值"]{}

  • 伪类选择器:选择一些特定状态的元素

    语法:元素:状态{}

    状态:

    ​ link:初始化状态

    ​ visited:被访问过的状态

    ​ active:正在访问的状态

    ​ hover:鼠标悬浮状态

3. CSS常用属性

3.1 字体文本

  • font-size：字体大小
  • color：文本颜色
  • text-align：文本对齐方式
  • line-height：行高
  • opacity：不透明度 取值范围为0.0-1.0 值越低越透明

3.2 背景

  • background-color：背景颜色

  • background-image：背景图片

    • 使用方式:background-image:url("url");

  • background-position:指定背景图像的位置

    • eg:background-position: right top

  • background-attachment:指定背景图像是固定的(fixed)还是滚动的(scroll).

3.3 边框

  • border-width:边框宽度;可以将宽度设置为特定大小（以 px、pt、cm、em 计），也可以使用以下三个预定义值之一：thin、medium 或 thick.

  • border-style:边框样式;一般使用solid-定义实线边框;dotted-定义点线边框;dashed-定义虚线边框;double-定义双边框;none-定义无边框;

  • border-color:定义边框颜色

  • 简写边框属性:border-width border-style(必须) border-color

    eg:border: 10px solid red

3.4 CSS盒子模型

  • 在CSS中可以使用盒子模型来控制每个块的布局

  • margin:外边距 以当前元素为参考向外的边距

  • padding:内边距 以当前元素为参考向内的边距

    • 默认情况下设置内边距会影响整个盒子的大小
    • 如果想设置内边距且固定盒子的width/height属性，需使用box-sizing: border-box;

  • float:可设置浮动(left/right)，可以让多个块级元素显示在同一行。
